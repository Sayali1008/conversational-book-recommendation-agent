# Phase 2 - Personalization with Collaborative Filtering

## Goal
1. Generate synthetic user-book interaction data.
1. Train CF model on implicit feedback.
1. Generate book embeddings (from Phase 1).
1. Implement hybrid scoring function with tunable α.
1. Add fallback for cold-start scenarios.
1. Evaluate with small test users and tweak α, latent factors, etc.
1. Wrap up as a working pipeline you can plug into a conversational interface.
